initial_structures:
Starting models used for simulation for each system studied (after adding hydrogens, solvent and Na+ ions in tLEaP).

input_files:
Amber input files used for the MM MD simulations, run in the following order:
min_1.in, min_2.in, heat.in, md_pro.in (run four times). 
Folders named after each system contain the restraint files for minimisation (min_restrains.rst) and heating (md_restrains.rst).  

parameter_files:
Force field parameter files for non-standard residue (S70 covalently bound with piperacillin). 
-	YPP.off  |  Amber20-compatible library file for atom types & charges. Partial charges derived using the contrained-charge derivation for non-standard residues as implemented in the R.E.D. Server.
-	YPP.frcmod  | Amber20-compatible force field modification file. The residue S70 was described using the Amber ff14SB force field, while piperacillin was parameterized using GAFF2. 
-	Mol-ia3_m1-c1.mol2  |  The optimized structure of the non-standard residue with partial charges generated by the R.E.D. Server.
